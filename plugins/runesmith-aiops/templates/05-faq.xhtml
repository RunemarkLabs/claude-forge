{COMPANY} - AIOPS FAQ

Why does my page publish fail with 400?

Confluence Cloud v2 does not accept markdown bodies. Skills convert to storage XHTML before publishing. If you're hand-rolling API calls, see confluence-format.md.

Why does my Jira create fail with 400 about assignee.name?

Jira Cloud needs accountId, not name. Use GET /rest/api/3/user/search?query=<email> to resolve.

Why does my page update return 409?

You posted a stale version.number. GET the page first, then PUT with number + 1.

Why does /rest/api/3/search not work anymore?

Deprecated on Cloud. Use POST /rest/api/3/search/jql.

How do I create a Confluence space via API?

v2 API does not expose space creation publicly. v1 POST /wiki/rest/api/space works but requires Confluence admin. For most users: create the space in the UI first, then run skills against it.

How do I find the right custom field id?

GET /rest/api/3/field lists every field including custom ones. Match by name. Cache the id in .credentials (e.g. ATLASSIAN_BUG_SEVERITY_FIELD).

Where do plugins install?

Cowork: ~/AppData/Roaming/Claude/.../rpm/plugin_*/. Claude Code: ~/.claude/plugins/. Marketplace install handles paths automatically; manual install is drag-drop into Cowork.

Where do I get help?

Run /devtools:help in your Claude client. Or read the marketplace README.
